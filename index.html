🔑 Подключение панели

Панель читает и пишет задачи прямо в твой GitHub-репозиторий antonzam5/assistant. Нужен токен доступа (один раз на устройство):

  1. Открой github.com → Fine-grained tokens → Generate new token
  2. Repository access → Only select repositories → assistant
  3. Permissions → Repository permissions → Contents → Read and write
  4. Expiration — максимум (1 год), Generate token, скопируй

Токен хранится только в этом браузере (localStorage) и даёт доступ только к этому репозиторию.
